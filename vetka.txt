Что такое ветки и для чего они нужны при работе с системой контроля версий.
Ветки это как черновик , нужны для совместных проектов .
Для создание ответвлений необходимо ввести команду git branch далее имя ветки
длятого что бы перемещатся по веткам необходимо ввести - git checkout 
Проверка на какой ветке мы находимся вводим git branch
